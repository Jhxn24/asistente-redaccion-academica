# Asistente de Redacción Académica

## Descripción

Aplicación web que permite analizar y mejorar textos académicos mediante inteligencia artificial. La herramienta recibe un texto como entrada y genera una versión revisada con mayor fluidez, variedad sintáctica y coherencia, manteniendo el significado original.

## Objetivo

Desarrollar una solución simple que ayude a mejorar la calidad de textos académicos utilizando modelos de lenguaje generativos.

## Tecnologías utilizadas

- Node.js
- Express.js
- Gemini API
- HTML
- CSS
- JavaScript

## Arquitectura

Usuario
↓
Frontend Web
↓
API Express
↓
Gemini API
↓
Texto revisado

## Funcionalidades

- Ingreso de texto académico
- Revisión automática mediante IA
- Generación de una versión mejorada
- Interfaz web sencilla

## Instalación

```bash
git clone https://github.com/tu_usuario/asistente-redaccion-academica.git

cd asistente-redaccion-academica

npm install

npm start
```
